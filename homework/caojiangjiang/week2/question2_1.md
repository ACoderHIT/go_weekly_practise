###多个变量的声明
第一种：
```
var x, y int
y, x = 5, 6
```

第二种
```
var y, x = 5, 6
```

第三种
```
y, x := 5, 6
```

第四种：
```
var ( // 这种因式分解关键字的写法一般用于声明全局变量
        x int
        y string
    )
    x = 8
    y = "gpwner"
```
